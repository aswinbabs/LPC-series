The 'MCBSTM32E Demo' project is an extensive demonstration program 
for the Keil 'MCBSTM32E' board.

This project is ported from ST STM3210E-EVAL board to Keil 
MCBSTM32E board.
 
For 'MCBSTM32E Demo' program there are available targets:
  MCBSTM32E:             ported example for Keil MCBSTM32E board
  MCBSTM32E Download MM: target used for downloading multimedia 
                         contents to SPI flash (!!! do not rebuild !!!)
